# AI Interview Simulator (Generative AI Project)

## Problem
Candidates lack realistic, adaptive interview practice with structured feedback.

## Solution
An LLM-powered interview simulator that:
- Generates role-specific interview questions
- Evaluates answers
- Provides actionable feedback

## Tech Stack
- Python
- Large Language Models (LLMs)
- Prompt Engineering
- Modular Agent Architecture

## Status
- Question Generator Agent implemented
- Evaluation and feedback agents in progress
